Etapa 1 - Linguagem C

  1. Tipos de Dados e Modificadores:
    • Tipos de Dados: Incluem os tipos primitivos como int (inteiros), float (ponto flutuante), char (caracteres) e double (ponto flutuante de dupla precisão).
    • Modificadores: Usados para alterar o tamanho ou o sinal dos tipos de dados básicos, como signed, unsigned, short e long.
  2. Operadores:
    • Operadores Aritméticos: +, -, *, /, %.
    • Operadores Relacionais: ==, !=, >, <, >=, <=.
    • Operadores Lógicos: &&, ||, !.
    • Operadores Bit a Bit: &, |, ^, ~, <<, >>.
    • Operadores de Atribuição: =, +=, -=, *=, /=, %=.
    • Operadores de Incremento e Decremento: ++, --.
  3. Variáveis Locais, Globais e Constantes:
    • Variáveis Locais: Declaradas dentro de funções ou blocos e só podem ser usadas dentro deles.
    • Variáveis Globais: Declaradas fora de todas as funções e podem ser acessadas por qualquer função no programa.
    • Constantes: Valores que não mudam durante a execução do programa, definidas usando 'const' ou '#define'.
  4. Estruturas condicionais em C:
    • if: Avalia uma condição e executa um bloco de código se a condição for verdadeira.
    • else: Executa um bloco de código se a condição no if for falsa.
    • else if: Avalia múltiplas condições em sequência.
    • switch: Seleciona um bloco de código para executar baseado no valor de uma variável.
  5. Estruturas de Repetição em C:
    • for: Laço de repetição com inicialização, condição e incremento.
    • while: Executa um bloco de código enquanto uma condição é verdadeira.
    • do while: Executa um bloco de código ao menos uma vez, e então repete enquanto a condição for verdadeira..
  6. Vetores e Matrizes:
    • Vetores (Arrays): Estruturas de dados que armazenam múltiplos valores do mesmo tipo usando um único nome.
    • Matrizes (Arrays Multidimensionais): Vetores de vetores, úteis para representar tabelas ou grades de dados.

Etapa 2 - Funções e Ponteiros

  1. Registros: Estruturas de dados compostas que podem conter variáveis de diferentes tipos. Utilizados para agrupar variáveis relacionadas sob um único nome.
  2. Manipulação de String: Operações básicas em strings, como cópia (strcpy), concatenação (strcat), comprimento (strlen), comparação (strcmp) e busca de caracteres (strchr).
  3. Passagem de Parâmetros por Referência: Envio de parâmetros para funções usando ponteiros, permitindo que a função modifique diretamente o valor da variável passada.
  4. Conceito de Ponteiros: Variáveis que armazenam endereços de memória. Usadas para manipular arrays, alocação dinâmica de memória, e para criar estruturas de dados complexas como listas ligadas.
  5. Funções em Linguagem C: Blocos de código reutilizáveis que executam uma tarefa específica. Incluem declaração, definição e chamada de funções, além de funções recursivas e anônimas.

Etapa 3 - Arquivos

  1. Recursividade: Técnica de programação onde uma função chama a si mesma para resolver subproblemas menores de um problema maior. Utilizada em algoritmos como a busca binária e a ordenação rápida (quicksort).
  2. Manipulação de Arquivo: Operações de leitura e escrita em arquivos, utilizando funções como fopen, fclose, fread, fwrite, fprintf, fscanf, fgets, e fputs. Inclui também a manipulação de arquivos binários e de texto.
  3. Resolução de Problemas Reais:Aplicação de conceitos de programação para desenvolver soluções práticas para problemas do mundo real, como gerenciamento de dados, automação de tarefas e desenvolvimento de software robusto e eficiente.
