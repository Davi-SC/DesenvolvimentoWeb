Etapa 1 - Linguagem de Marcação

  1. Linguagens de marcação para criação de páginas para web:
    • Definição: Linguagens como HTML e XML são usadas para estruturar e apresentar conteúdo na web. HTML é a base de todas as páginas web, definindo elementos como títulos, parágrafos, links e imagens.
    • Uso de Aplicativos de Suporte à Criação de Páginas: Ferramentas como editores de texto (Sublime Text, VSCode) e construtores de sites (WordPress, Wix) auxiliam na criação de páginas web, oferecendo funcionalidades para edição e visualização do código.

Etapa 2 - Linguagem de Estilo em Cascata

  1. Conceitos de Usabilidade, Cores, Estrutura de Navegação e Publicação na Internet:
    • Usabilidade: Refere-se à facilidade com que os usuários podem navegar e interagir com um site. Inclui aspectos como clareza, eficiência e satisfação.
    • Cores: A escolha de cores afeta a aparência e a acessibilidade de um site. Cores complementares e contrastantes são usadas para destacar elementos importantes.
    • Estrutura de Navegação: A organização dos links e menus em um site deve ser intuitiva e lógica para facilitar a navegação dos usuários.
    • Publicação na Internet: Envolve o processo de tornar o site acessível na web, geralmente por meio de um servidor web ou serviço de hospedagem.
  2. Folhas de Estilo em Cascata (CSS):
    • Definição: CSS é usado para controlar a aparência e o layout das páginas web. Permite a separação entre o conteúdo (HTML) e o design, facilitando a manutenção e a atualização do site.
    • Seletores e Propriedades: CSS utiliza seletores para aplicar estilos a elementos HTML específicos. Propriedades como cor, margem, padding, e fonte são usadas para definir a aparência dos elementos.
  3. Framework Responsivo para Estilização de Páginas Web:
    • Definição: Frameworks como Bootstrap e Foundation fornecem estilos e componentes prontos para criar sites responsivos, que se adaptam a diferentes tamanhos de tela e dispositivos.
    • Componentes: Incluem grades responsivas, botões, formulários, navegação e outros elementos de interface que facilitam o desenvolvimento de sites modernos e responsivos.

Etapa 3 - Linguagem Dinâmica

  1. JavaScript:
    • Definição: JavaScript é uma linguagem de programação utilizada para criar interatividade e funcionalidades dinâmicas em páginas web. É executado no navegador do usuário, permitindo a manipulação do DOM, validação de formulários, animações e muito mais.
    • Fundamentos: Incluem variáveis, operadores, estruturas condicionais, loops e funções. JavaScript também suporta programação orientada a objetos e eventos.
  2. Desenvolvimento de Páginas Web:
    • Integração: JavaScript é frequentemente usado junto com HTML e CSS para criar experiências de usuário ricas e interativas. Frameworks e bibliotecas como jQuery, React, e Angular ampliam as capacidades do JavaScript.
    • Práticas de Desenvolvimento: Incluem o uso de boas práticas de codificação, controle de versão, testes e depuração para garantir que o código JavaScript seja eficiente, legível e livre de erros.
